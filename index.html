1. JavaScript Introduction

Variables - var, let or const.

Data Types Primitives (value types) and Non primitives( Reference types)

Primitives are - string, number, boolean, null, undefined. Non-primitives are Objects and Symbols.

Boolean Booleans have true and false values. In JS, 0, " ". null, undefined, false and NAN are falsey values. Use !! to check if a value is truthy or falsey.

Arrays is a list to store values. Arrays are reference types. typeOf array is Object.

Array Methods

  • push - Adding element towards the end of the array
  • unshift - Adding elements towards the beginning of the array
  • pop - removes element from the end of the array
  • shift - removes element from the beginning of the array
  • splice - splice is a powerful generalization of push, pop, unshift, and shift all in one! It takes atleast 2 arguments. Starting index, how many elements needs to be removed, the new elements to add.

more Array Methods

  • Slice - makes copy of the array
  • indexOf - useful to check if an element is in array or not.
  • lastIndexOf - checks from the end of the array.
  • concat - joins two arrays.
  • join - seperates them by a delimeter.
